===============
MIS Report Mail
===============

This module allows to send MIS Reports by email.


Installation
============

Just install.


Configuration
=============

No configuration needed.


Usage
=====

Go to Invoicing -> Reporting -> MIS Reports, choose a report and press the "Send Mail" button. Add the recipients and send it.
